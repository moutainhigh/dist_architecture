# 通用分布式系统架构
## 项目说明  
比较通用的互联网系统架构，使用Java作为后端语言，Vue作为前端语言，后端基础框架使用SpringBoot+Dubbo来实现微服务，前端则使用ElemtneUI实现前后端分离，使用到功能组件包括RocketMQ、Redis、Elasticsearch、FastDFS、Zookeeper、Nacos、Quartz、SpringCloudGateway、JavaMail 等
